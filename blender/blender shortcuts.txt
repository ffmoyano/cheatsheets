NEW INSTALL

defaults: spacebar -> search
addons: textools (instalar) mesh looptools (activar) node wrangler (activar)


NAVIGATION

n - abre la ventana de transformación del objeto seleccionado
t - muestra u oculta la barra de herramientas
pulsar rueda raton + arrastrar - girar vista
maysc + pulsar rueda raton - mover vista sin girar
mover rueda raton - zoom
num 5 - cambia entre vista ortográfica (vista aplanada del objeto) y vista en perspectiva
num 1 - vista ortografica frontal
num 3 - vista lado derecho
num 7 - vista desde arriba
ctrl + num 1 - vista desde atras
ctrl + num 3 - vista desde lado izquierdo
ctrl + num 7 - vista desde abajo
h - ocultar objeto seleccionado
alt h - desocultar todos los objetos ocultos
g - mover objeto (+ x, y, z para mover solo en ese eje) (+ shift x,y,z para mover por los dos ejes no seleccionados) 
s - escalar objeto (+ x, y, z para escalar solo ese eje) (+ shift x,y,z para escalar los dos ejes no seleccionados)
r - rotate objeto (+ x, y, z para rotar por ese eje) (+ shift x,y,z para rotar por los dos ejes no seleccionados)
mientras se transforma, click izquierdo para confirmar, click derecho para cancelar
mientras se transforma si pulsamos x, y o z dos veces en lugar de una, hara la transformación en el espacio local(relativo a la orientacion del objeto) en lugar del espacio global.
alt + transform key - elimina los cambios hechos al objeto y lo reinicia en base al espacio global

ctrl + z - deshacer
ctrl + shift + z - rehacer

shift + a - abrir menu de añadir objeto

shift + right click - mover el cursor 3d (donde se originan nuevos objetos)

shift + c - devolver el cursor 3d al origen del mundo (0,0,0) y mueve la vista a dicho origen

shift + d - duplica el objeto seleccionado

f2 - renombrar el objeto seleccionado (o el ultimo que estuvo seleccionado)

supr - elimina el objeto seleccionado

x - elimina el objeto seleccionado con dialogo de confirmacion

shift + click izquierdo - seleccionar algo sin perder la seleccion anterior

select multiple + ctrl + j - unir los objetos seleccionados

z - abre el menu de modos de shading

tab - cambiar entre edit mode y object mode

w - cambia entre los distintos modos de seleccion

ctrl +tab - menu de modos de edicion

EDIT MODE

1 - select vertices
2 - select lines
3 - select faces

alt + click - loop select, selecciona todos los vertices/lines/faces conectados horizontalmente o verticalmente al nodo seleccionado

alt + q - cambia el objeto activo a aquel sobre el que el raton esta haciendo hovering (no hace nada si el raton no está sobre un objeto)

select node y luego ctrl + click en otro nodo - shortest path selection, selecciona el camino mas corto de nodos necesario para llegar al primer nodo

o - togglear edicion proporcional

a - seleccionar todos los nodos del objeto editado

shift + s - abrir menu de ajuste (snapping menu)


SCULPT MODE 

f - seleccionar tamaño pincel
